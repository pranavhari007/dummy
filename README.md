# 🎾 PlayNow - Sports Venue Booking & Community

PlayNow is a modern, full-stack web application designed for sports enthusiasts to discover, book, and host matches at local sports venues. Whether it's football, badminton, or cricket, PlayNow brings the community together through seamless booking and social features.

## ✨ Key Features

- **🏠 Interactive Home Page**: Stunning dark-themed UI with rich animations and featured venues.
- **🛡️ Verified Authentication**: Secure login/signup with 10-digit mobile verification and OTP logic.
- **🏟️ Venue Management**: Browse venues, view detailed availability, and see property highlights.
- **📅 Smart Booking**: Easy slot selection with instant feedback and price calculations.
- **📱 Social Feed**: A dedicated sports feed where you can post moments, follow friends, like, and comment.
- **🤝 Host a Match**: Split booking costs with the community by hosting open matches.
- **📊 User & Owner Dashboards**: Dedicated interfaces for players to manage bookings and owners to manage properties.
- **❌ Smart Cancellation**: Automated refund calculations based on the time remaining before the match.
- **📱 Mobile Responsive**: Fully optimized for a premium experience on any device.

## 🛠️ Technology Stack

- **Frontend**: React.js, Vite, Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Atlas).
- **State Management**: React Context API.

## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/playnow.git
   cd PlayNow
   ```

2. **Frontend Setup**
   ```bash
   npm install
   npm run dev
   ```

3. **Backend Setup**
   ```bash
   cd backend
   npm install
   node server.js
   ```
